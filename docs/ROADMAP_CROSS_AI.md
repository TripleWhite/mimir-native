# Mimir Phase 3: 跨 AI 平台记忆共享

**周期**: 30 天  
**目标**: 实现用户与不同 AI 对话的跨平台记忆共享

---

## 🎯 核心场景

### 场景 1: 跨平台上下文继承
```
周一 - Claude:
用户: "我想做一个用户系统，基于微服务架构"
Claude: "好的，建议使用 JWT 认证..."
→ Mimir 存储: [用户系统, 微服务, JWT, Claude]

周三 - ChatGPT:
用户: "帮我写这个用户系统的代码"
ChatGPT: "什么用户系统？"
用户: "就是周一和 Claude 讨论的那个"
ChatGPT: "哦，基于微服务 + JWT 的那个..."
→ Mimir 检索 → 注入上下文 → 连续对话
```

### 场景 2: 跨平台计划追踪
```
周五 - Claude:
用户: "帮我规划下周工作：完成用户系统，开始支付模块"
→ Mimir 存储: [待办: 用户系统, 支付模块, 时间: 下周]

下周二 - ChatGPT:
用户: "我上周规划的工作进展如何？"
ChatGPT: "你计划上周完成用户系统和开始支付模块..."
```

### 场景 3: 跨平台风格一致性
```
Claude 对话中:
用户: "我喜欢简洁的代码风格，不要过多注释"
→ Mimir 存储: [偏好: 简洁代码, 少注释]

ChatGPT 对话中:
ChatGPT 自动生成代码时:
"根据你之前的偏好，我使用简洁风格..."
```
