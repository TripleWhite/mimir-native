# Mimir Phase 3: Mimir Agent - 统一 AI 入口

**愿景**: 用户只需要和 Mimir 对话，Mimir 自动调度其他 AI

---

## 🎯 核心交互流程

### 场景 1: 用户直接问 Mimir
```
用户: "帮我写用户系统代码"

Mimir Agent:
1. 检索记忆: "找到你之前和 Claude 讨论的微服务架构..."
2. 组织 Prompt: "基于之前的 JWT + 微服务设计，写代码..."
3. 调用 Claude: 发送优化后的完整上下文
4. 返回给用户: "Claude 生成的代码如下..."
```

### 场景 2: Mimir 主动介入
```
用户正在 Claude 对话: "继续那个项目"
Claude: "哪个项目？"

Mimir Agent 检测:
1. 分析对话: 用户提到"继续"但没有上下文
2. 检索记忆: 找到之前讨论的用户系统
3. 发送提示: "你在 Mimir 中记录过用户系统项目，要注入吗？"
4. 用户确认 → 自动注入上下文
```
